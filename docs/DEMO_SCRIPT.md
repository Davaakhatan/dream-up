## Visual Cues:

- **Use screen recordings** of actual tests running
- **Highlight important UI elements** with subtle animations
- **Show real game examples** (2048, Pac-Man, etc.)
- **Display actual test results** (not mockups)
- **Show the browser automation** in action (if using visible browser mode)

---

## Tips for Recording:

1. **Use a clean browser** - no extensions, bookmarks visible
2. **Record at 1080p or higher** for clarity
3. **Use smooth mouse movements** - avoid jerky motions
4. **Add subtle zoom-ins** on important UI elements
5. **Keep background music** low and professional
6. **Add captions** for accessibility
7. **Show actual URLs** so viewers can try themselves
